	Fonctionnement du programme
1) Le fichier "text_matrice.txt" doit etre structuré comme suite: 
	-La premiere ligne contient le nombre de noeuds suivi du nombre d'arretes;
	-Les autres lignes contiennent respectivement les differentes arretes suivi de leur poids. 
Les arretes doivent etre organisée en partant de la source vers le puits.
2) Ce programme peut etre compilé/executé sous un IDE ou par ligne de commande en tapant "g++ -o TP_RO TP_RO.cpp -lm"
////
	Un second programme faite en java permet d'obtenir le flot max mais ne permet pas d'importer la matrice dans un fichie
3) Le programme FordFulkerson.java peut etre executé par ligne de commande comme suite: 
	-javac FordFulkerson.java
	-java FordFulkerson
Ensuite entrer le nombre de noeuds et valider
Enfin vous devrez fournir la matrice du graph et valider pour obtenir le flot max.
